HTML5 - FIGCAPTION

The purpose of a caption is to add a visual explanation to an image.

            <figure>
            <img src="/images/car.jpg" alt="Car" width="304" height="228">
            <figcaption>Fig.1 - Hot wheels.</figcaption>
            </figure>


HTML5 - NAV

The nav element is intended for large blocks of navigation links. However, not all links in a document should be inside a element!

HTML | CSS | JavaScript | jQuery 
            <nav>
            <a href="/html/">HTML</a> |
            <a href="/css/">CSS</a> |
            <a href="/js/">JavaScript</a> |
            <a href="/jquery/">jQuery</a>
            </nav>
