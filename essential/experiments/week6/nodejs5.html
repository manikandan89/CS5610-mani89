Node JS - Update and Delete

This Web Page is mainly about the update and delete operation

APP.PUT - SERVER

The put operation can be performed to update a particular element

                        app.put("/api/players/:id", function (req, res) {
                        var index = req.params.id;
                        players[index] = req.body;
                        res.json(players);
                        });

APP.DELETE - SERVER

The delete operation can be performed to delete a particular element

                        app.delete("/api/players/:id", function (req, res) {
                        Player.remove({ _id: req.params.id }, function (err,doc) {
                            Player.find(function (err, docs) {
                                res.json(docs);
                            });
                        })
                        });

CONTROLLER

The controller calls the restful put and delete of the node js server respectively

It sends the response back to the client

                         $scope.update = function (player) {
                            $http.put("/api/players/" + $scope.selectedIndex, player)
                             .success(function (response) {
                                 $scope.players = response;
                             });
                        }


                        $scope.remove = function (id) {
                        $http.delete("/api/players/" + id)
                        .success(function (response) {
                            $scope.players = response;
                        });
                         }

CLIENT

The client side functionality deletes or updates the contents when clicked on the repective button

This invokes the controller with a call to get()

                           <button class="btn btn-primary pull-right" ng-click="update(player)">
                            <span class="glyphicon glyphicon-ok"></span>
                        </button>
                        <button class="btn btn-danger pull-right" ng-click="remove(player._id)">
                            <span class="glyphicon glyphicon-remove"></span>
                        </button>
