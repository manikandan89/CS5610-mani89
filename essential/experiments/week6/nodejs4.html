Node JS - Get

This Web Page is mainly about the get operation

APP.GET - SERVER

This experiment demonstrates the implementation of the get all and get by id

The get operation can be performed to retrieve all the elements

It can also retrieve a particluar element by passing the required parameter

                           app.get("/api/players", function (req, res) {
                            Player.find(function (err, docs) {
                                res.json(docs);
                            });
                        });


                        app.get("/api/players/:id", function (req, res) {
                            Player.findById(req.params.id,function (err, doc) {
                                res.json(doc);
                            });
                        });

CONTROLLER

The controller calls the restful get of the node js server

It sends the response back to the client

                          $http.get("/api/players")
                            .success(function (response) {
                                $scope.players = response;
                            });

CLIENT

The client side functionality displays the contents

This invokes the controller with a call to get()

                           <tr ng-repeat="player in players">
                            <td>{{player.name}}</td>
