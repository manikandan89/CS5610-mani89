This is a form element form for a login form Each form element holds the Label for the input element. 

            <div>
            <label for="name">Username</label>
            <input id="name" type="text" placeholder="Username">
            </div>
            <div>
            <label for="password">Password</label>
            <input id="password" type="password" placeholder="Password">
            </div>
            <div>
            <button type="submit" class="pure-button pure-button-primary">Submit</button>
            </div>

Since designing only the form element is not good enough, I have applied more cascading to the text and Label Elements. Basically i have applied the -webkit-box-shadow: 0px 0px 8px rgba(0, 0, 0, 0.3); which gives a shadowing effect to the test fields. 

        input {
            width: 200px;
            display: block;
            border: 1px solid #999;

            -moz-box-shadow: 0px 0px 8px rgba(0, 0, 0, 0.3);
