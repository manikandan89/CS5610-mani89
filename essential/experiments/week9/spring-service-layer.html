Spring - Service

This Web Page is mainly about the Spring service

All the business logic should be performed in the service layer and not in the controller.

                      @Service
                            @Transactional
                            public class HomeServiceImpl implements HomeService {

                                @Override
                                public Employee findEmployee() {
                                    // TODO Auto-generated method stub
                                    Employee person = new Employee();
                                    person.setId(1);
                                    person.setFirstName("Sritharan");
                                    person.setLastName("Mahendra Babu");
                                    return person;
                                }
                            }

This service is autowired with the controller

                    private HomeService homeService;
    
                         @Autowired
                            public HomeController(HomeService homeService) {
                                this.homeService = homeService;
                            }
