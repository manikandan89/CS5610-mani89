Spring - JSON Consumption

This Web Page is mainly about the rendering of a json object through Spring Jackson API

The controller calls the corresponding handler according the request url. The JSON object which is constructed internally through the business logic is returned by Response Body

                       @RequestMapping("/view")
                         public @ResponseBody Employee getPerson() {
                            Employee person = new Employee();
                            person.setId(1);
                            person.setFirstName("Rafael");
                            person.setLastName("Nadal");
                            return person;
                        }

I am using the jackson codehaus and json simple parser dependencies in Maven to handle JSON objects for project. Through this demo I understood how json object can be rendered. I will use this in my project to render the contents of json using Angular JS
