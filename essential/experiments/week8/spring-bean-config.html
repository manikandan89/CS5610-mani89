Spring - Bean Configuration

This Web Page is mainly about spring bean configuration

WEB.XML

The web.xml is the root for all the spring config beans. The servlet and its mapping is defined here.

                        <context-param>
                        <param-name>contextConfigLocation</param-name>
                        <param-value>/WEB-INF/spring/root-context.xml,
                                     /WEB-INF/spring/security-config.xml
                        </param-value>
                        </context-param>
                        <servlet-name>appServlet</servlet-name>
                            <servlet-class>org.springframework.web.servlet.DispatcherServlet</servlet-class>
                        <servlet-mapping>
                            <servlet-name>appServlet</servlet-name>
                            <url-pattern>/</url-pattern>
                        </servlet-mapping>

DISPATCHER-SERVLET

The servlet.xml handles the config for spring mvc. It maintains the resources and the corresponding packages to be scanned during a webservice call

                        <resources mapping="/resources/**" location="/resources/" />
                        <beans:bean class="org.springframework.web.servlet.view.InternalResourceViewResolver">
                        <beans:property name="prefix" value="/WEB-INF/views/" />
                        <beans:property name="suffix" value=".jsp" />
                        </beans:bean>
                        <context:component-scan base-package="edu.neu.project" />
