This Web page demonstrates my learning on Javascript DOM Objects

I have started with the Javascript Web Design. The implementation is the Slide In of the Head and the Login Box. Basically the Login form element slides In from the right to the center of the Web Page. 

            function setup() {
            var e = document.getElementById("LoginBox");
            e.className = "slidein";
        }

The above javascript function starts onload of the Body content. The function receives the Login element and works on the DOM element which slides in from the right to the center.
