Javascript Experimentation - 5

This experiment is intended to show how javascript changes HTML.



The button's onclick event is given the "getTime()" function, and then the place holder with id "time"'s innerHTML will show current time:

                     function getTime()
                     {
                        document.getElementById("time").innerHTML = Date();
                    }



Hello!

Similarly, the style can also be changed:

                  function changeText() {
                    txt = document.getElementById("text");
                    if (txt.innerHTML.match("Hello")) {
                    txt.innerHTML = "Good Bye!";
                    txt.style.background = "lightcoral";
                    }
                    else {
                    txt.innerHTML = "Hello!";
                    txt.style.background = "lightyellow";
                     }
                    }


The image's source link can also be changed:

                 function changeImage() {
                element = document.getElementById('img');
                if (element.src.match("federer")) {
                element.src = "/images/nadal.jpg";
                }
                 else {
                element.src = "/images/federer.jpg";
                }
                }
