Browser object confirm() in javascript

It displays the confirm dialog box. It has message with ok and cancel buttons.




				function msgConfirm(){  
				var v= confirm("Are u sure?");  
				if(v==true){  
				alert("ok");  
				}  
				else{  
				alert("cancel");  
				}


Browser object prompt() in javascript

It displays prompt dialog box for input. It has message and textfield.




			function msgPrompt(){  
				var v= prompt("Who are you?");  
				alert("I am "+v);  
  
				}


Browser object open() in javascript

It displays the content in a new window.




		function msgOpen(){

Browser object setTimeout() in javascript

It performs its task after the given milliseconds.




		function msgTimeout(){  
			setTimeout(  
			function(){  
				alert("Welcome to Javatpoint after 2 seconds")  
			},2000);  
  
			}
